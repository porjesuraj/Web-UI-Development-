10. Check whether it is Armstrong no. (when sum of cube of all digits of equals the number then the number is called as Armstrong number) Example: 153 (1 * 1 * 1)+(5 * 5 * 5)+(3 * 3 * 3) = 1 + 125 + 27 = 153 Input: 936 Output: 936 is not an Armstrong number Input: 153 Output: 153 is an Armstrong number
