 # CSS Position 
  1. Static
   - default position
   - decided by code structure
   - top,left,bottom,and right properties will be ignored
   - e.g.
    ```css
     button {
            position:static;
            }
     ```
  2. Relative
    - the new position (by setting top,bottom,left and right) with
    respectto default position.
    - e.g
      ```css
      button
      {
        position:relative;
        top="20px";
        left="20px";
      }  
      ```
